Instagram'da profesyonel görünmek için doğru boyutlar şart. Yanlış boyutlar kırpılmış, bulanık veya amatör görüntülere neden olur. İşte 2026 için güncel Instagram boyutları.

![Instagram Görsel Boyutları Rehberi](/blog/instagram_sizes_cover.png)

## Instagram Post Boyutları

### Kare Post (Square)

En klasik ve güvenli format.

| Özellik | Değer |
|---------|-------|
| Boyut | 1080 x 1080 px |
| En-boy oranı | 1:1 |
| Dosya formatı | JPG, PNG |
| Max dosya boyutu | 30 MB |

**Ne zaman kullanılır:**
- Ürün fotoğrafları
- Grafik tasarımlar
- Alıntılar

### Dikey Post (Portrait)

Feed'de daha fazla alan kaplar.

| Özellik | Değer |
|---------|-------|
| Boyut | 1080 x 1350 px |
| En-boy oranı | 4:5 |

**Ne zaman kullanılır:**
- Ürün detay fotoğrafları
- Model çekimleri
- İnfografikler

### Yatay Post (Landscape)

| Özellik | Değer |
|---------|-------|
| Boyut | 1080 x 566 px |
| En-boy oranı | 1.91:1 |

**Ne zaman kullanılır:**
- Panoramik görüntüler
- Before/after karşılaştırmaları

## Instagram Reels Boyutları

Reels, Instagram'ın en yüksek erişim sağlayan formatı.

| Özellik | Değer |
|---------|-------|
| Boyut | 1080 x 1920 px |
| En-boy oranı | 9:16 |
| Süre | 15-90 saniye |
| Dosya formatı | MP4, MOV |

### Reels İçin İpuçları

**Güvenli alan:**
- Üst 250px: Profil bilgisi
- Alt 350px: Açıklama ve butonlar
- Önemli içerik ortada olmalı

**E-ticaret için Reels fikirleri:**
- Ürün tanıtımı
- Paketleme videoları
- Before/after
- Nasıl kullanılır
- Müşteri yorumları

## Instagram Story Boyutları

| Özellik | Değer |
|---------|-------|
| Boyut | 1080 x 1920 px |
| En-boy oranı | 9:16 |
| Süre | Max 60 saniye |
| Görünürlük | 24 saat |

### Story Güvenli Alanlar

```
┌─────────────────────┐
│    Üst: 250px       │  ← Dokunmayın (profil bilgisi)
│                     │
│   ┌─────────────┐   │
│   │  Güvenli    │   │  ← İçerik buraya
│   │    Alan     │   │
│   └─────────────┘   │
│                     │
│    Alt: 350px       │  ← Dokunmayın (mesaj alanı)
└─────────────────────┘
```

| Özellik | Değer |
|---------|-------|

| Görüntülenme | 110 x 110 px (circular) |
| Format | JPG, PNG |

## Instagram Carousel (Çoklu Görsel)

Birden fazla görsel paylaşımı.

| Özellik | Değer |
|---------|-------|
| Boyut | 1080 x 1080 px veya 1080 x 1350 px |
| Max görsel | 10 adet |
| Mix format | Görsel + video olabilir |

**E-ticaret için carousel fikirleri:**
1. Ürün koleksiyonu
2. Özellik anlatımı (slide by slide)
3. Müşteri yorumları serisi
4. How-to rehberi
5. Ürün karşılaştırması

## E-Ticaret için Instagram Görselleri

### Ürün Fotoğrafı Checklist

- [ ] Beyaz veya nötr arka plan
- [ ] Yeterli aydınlatma
- [ ] Yüksek çözünürlük (min 1080px)
- [ ] Birden fazla açı
- [ ] Ölçek referansı (gerekirse)
- [ ] Marka tutarlılığı

### Instagram'da Satış Yapan Görsel Türleri

| Görsel Tipi | Engagement | Satış Etkisi |
|-------------|------------|--------------|
| Lifestyle (kullanımda) | Yüksek | Yüksek |
| Flat lay | Orta | Orta |
| Ürün detay | Düşük | Yüksek |
| User-generated | Çok yüksek | Çok yüksek |

## Sık Yapılan Hatalar

1. **Düşük çözünürlük** - Min 1080px kullanın
2. **Yanlış kırpma** - Güvenli alanları bilin
3. **Aşırı metin** - Görsel ağırlıklı olsun
4. **Tutarsız stil** - Marka kimliği koruyun
5. **Watermark spam** - Minimal logo kullanın
