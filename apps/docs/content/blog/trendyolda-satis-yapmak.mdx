## Ürün Listeleme

### Başarılı Listeleme İçin

**Ürün başlığı:**
- Anahtar kelimeleri içermeli
- Marka + Ürün + Özellik formatı
- Max 150 karakter

```
✓ İyi: "Nike Air Max 90 Erkek Spor Ayakkabı Beyaz 42 Numara"
✗ Kötü: "Spor ayakkabı erkek"
```

**Ürün açıklaması:**
- Minimum 100 kelime
- Özellikler madde madde
- SEO uyumlu içerik

**Görseller:**
- Minimum 3 görsel
- Beyaz arka plan
- Yüksek çözünürlük (min 1000x1000)
- Farklı açılar



### Varyant Yönetimi

Renk, beden gibi varyantları doğru ekleyin:

```
Ana ürün: T-Shirt
├── Renk: Beyaz
│   ├── Beden: S
│   ├── Beden: M
│   └── Beden: L
└── Renk: Siyah
    ├── Beden: S
    ├── Beden: M
    └── Beden: L
```

## Fiyatlandırma Stratejisi

### Fiyat Belirleme Formülü

```
Satış Fiyatı = (Maliyet + Kar) / (1 - Komisyon Oranı)

Örnek:
Maliyet: 100 TL
Hedef kar: 50 TL
Komisyon: %18

Satış Fiyatı = (100 + 50) / (1 - 0.18) = 183 TL
```

### Rekabetçi Fiyatlama

- Rakip fiyatlarını takip edin
- Fiyat alarmları kurun
- Kampanya dönemlerinde marjı düşürün
