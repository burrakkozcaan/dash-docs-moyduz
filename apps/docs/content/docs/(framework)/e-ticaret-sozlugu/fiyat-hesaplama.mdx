## Fiyatlandırma Temelleri

E-ticarette doğru fiyatlandırma, karlılığın temelidir. Tüm maliyetleri hesaba katarak fiyat belirlemelisiniz.

---

| Kalem | Ortalama Oran |

| Reklam | %5-15 |

---

## Fiyat Hesaplama Formülü

```
Satış Fiyatı = (Maliyet + Hedef Kar) / (1 - Toplam Kesinti Oranı)
```

### Örnek Hesaplama

**Ürün bilgileri:**
- Alış fiyatı: 100 TL
- Hedef kar marjı: %30
- Sanal POS: %2
- Kargo: %8
- Reklam: %10

**Hesaplama:**

```
Toplam kesinti = %2 + %8 + %10 = %20
Hedef kar = 100 × 0.30 = 30 TL
Satış fiyatı = (100 + 30) / (1 - 0.20) = 162.5 TL
```

**KDV dahil fiyat (%20 KDV):**

```
KDV dahil = 162.5 × 1.20 = 195 TL
```

---

## Kar Marjı Türleri

### Brüt Kar Marjı

```
Brüt Kar = Satış Fiyatı - Ürün Maliyeti
Brüt Kar Marjı = (Brüt Kar / Satış Fiyatı) × 100
```

### Net Kar Marjı

```
Net Kar = Satış Fiyatı - Tüm Maliyetler
Net Kar Marjı = (Net Kar / Satış Fiyatı) × 100
```

## Psikolojik Fiyatlandırma

### Tek Sayı Kuralı

```
❌ 200 TL
✓ 199 TL veya 199.90 TL
```

### Karşılaştırmalı Fiyatlandırma

```
İndirimli fiyat gösterimi:
Eski fiyat: 299 TL (üstü çizili)
Yeni fiyat: 199 TL
İndirim: %33
```

## Hazır Platform vs Özel Yazılım

| Fiyatlandırma Özelliği | Hazır Platform | Özel Yazılım |
|------------------------|---------------|--------------|
| Müşteriye özel fiyat | Sınırlı/Yok | Tam destek |

## Önerilen Araçlar

### Excel/Sheets Şablonu

Basit fiyat hesaplama için formül:

```
=((Maliyet*(1+KarMarji))/(1-ToplamKesinti))*(1+KDV)
```

### Özel Yazılım Avantajı

- Otomatik maliyet güncelleme
- Dinamik kar marjı hesaplama
- Rakip fiyat takibi
- Kampanya simülasyonu
