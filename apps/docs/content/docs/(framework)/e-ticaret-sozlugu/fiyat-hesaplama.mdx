Alış fiyatı 100 TL olan bir ürünü kaç TL'ye satmanız gerekiyor?

Çoğu yeni satıcı 130-140 TL diyor. Gerçek cevap 162,5 TL — ve KDV dahil 195 TL.

Aradaki fark komisyon, kargo ve reklam maliyetlerinin yanlış hesaplanmasından kaynaklanıyor. Bu rehberde doğru formülü ve gerçek kategori marjlarını gösteriyoruz.

## Fiyatlandırma Temelleri

E-ticarette doğru fiyatlandırma, karlılığın temelidir. Tüm maliyetleri hesaba katarak fiyat belirlemelisiniz.

## Maliyet Kalemleri

### Ürün Maliyeti

| Kalem | Açıklama |
|-------|----------|
| Alış fiyatı | Tedarikçiden satın alma |
| Nakliye | Deponuza geliş maliyeti |
| Gümrük | İthalatta vergiler |
| Depolama | Stok tutma maliyeti |

### Operasyonel Maliyetler

| Kalem | Ortalama Oran |
|-------|---------------|
| Kargo | %5-10 |
| Paketleme | %1-3 |
| İade | %2-5 |
| Personel | Değişken |

### Platform/Satış Maliyetleri

| Kalem | Ortalama Oran |
|-------|---------------|
| Sanal POS | %1.5-3 |
| Marketplace komisyon | %8-25 |
| Reklam | %5-15 |
| Hosting/Platform | Sabit |

## Fiyat Hesaplama Formülü

```
Satış Fiyatı = (Maliyet + Hedef Kar) / (1 - Toplam Kesinti Oranı)
```

### Örnek Hesaplama

Ürün bilgileri:
- Alış fiyatı: 100 TL
- Hedef kar marjı: %30
- Sanal POS: %2
- Kargo: %8
- Reklam: %10

Hesaplama:
```
Toplam kesinti = %2 + %8 + %10 = %20
Hedef kar = 100 × 0.30 = 30 TL
Satış fiyatı = (100 + 30) / (1 - 0.20) = 162.5 TL
```

KDV dahil fiyat (%20 KDV):
```
KDV dahil = 162.5 × 1.20 = 195 TL
```

## Kar Marjı Türleri

### Brüt Kar Marjı

```
Brüt Kar = Satış Fiyatı - Ürün Maliyeti
Brüt Kar Marjı = (Brüt Kar / Satış Fiyatı) × 100
```

### Net Kar Marjı

```
Net Kar = Satış Fiyatı - Tüm Maliyetler
Net Kar Marjı = (Net Kar / Satış Fiyatı) × 100
```

## Kategori Bazlı Kar Marjları

| Kategori | Ortalama Brüt Kar | Ortalama Net Kar |
|----------|-------------------|------------------|
| Elektronik | %10-20 | %5-10 |
| Giyim | %40-60 | %15-25 |
| Kozmetik | %50-70 | %20-35 |
| Gıda | %15-30 | %5-15 |
| Mobilya | %30-50 | %10-20 |
| Evcil Hayvan | %30-45 | %18-32 |

<Callout type="warning">
**Net marj gerçekten önemli.** Giyimde %50 brüt marj çekici görünür, ama %25 iade oranı + %15 reklam gideri + %12 kargo ile net marj %8-12'ye düşer. Kategoriye girişten önce gerçek net marjı hesaplayın.
</Callout>

## Psikolojik Fiyatlandırma

### Tek Sayı Kuralı

```
❌ 200 TL
✓ 199 TL veya 199.90 TL
```

### Karşılaştırmalı Fiyatlandırma

```
Eski fiyat: 299 TL (üstü çizili)
Yeni fiyat: 199 TL
İndirim: %33
```

```
=((Maliyet*(1+KarMarji))/(1-ToplamKesinti))*(1+KDV)
```
