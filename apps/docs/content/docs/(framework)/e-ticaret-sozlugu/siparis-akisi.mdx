## Sipariş Akışı Nedir?

Sipariş akışı, müşterinin sepete ürün eklemesinden teslimatın tamamlanmasına kadar geçen tüm süreçtir.

---

## B2C Sipariş Akışı

### Standart Akış

```
Sepete Ekle
    ↓
Sepet Görüntüle
    ↓
Checkout (Ödeme Sayfası)
    ↓
Adres Bilgileri
    ↓
Kargo Seçimi
    ↓
Ödeme
    ↓
Sipariş Onayı
    ↓
Hazırlık
    ↓
Kargoya Teslim
    ↓
Teslimat
```

### Sipariş Durumları

| Durum | Açıklama |
|-------|----------|
| Beklemede | Ödeme bekleniyor |
| Onaylandı | Ödeme alındı |
| Hazırlanıyor | Paketleme aşamasında |
| Kargoda | Kargo firmasına teslim |
| Teslim Edildi | Müşteriye ulaştı |
| İptal | Sipariş iptal edildi |
| İade | İade süreci başladı |

---

## B2B Sipariş Akışı

B2B'de süreç daha karmaşıktır:

### Onay Mekanizmalı Akış

```
Sepete Ekle
    ↓
Teklif İste (Opsiyonel)
    ↓
Teklif Onayı
    ↓
Sipariş Oluştur
    ↓
Kredi Limit Kontrolü
    ↓
Yönetici Onayı (Tutar bazlı)
    ↓
Sipariş Onayı
    ↓
Hazırlık
    ↓
Sevkiyat
    ↓
Fatura Kesimi
    ↓
Teslimat
```

### B2B Onay Kuralları

| Sipariş Tutarı | Onay Gereksinimi |
|---------------|------------------|
| 0-10.000 TL | Otomatik |
| 10.000-50.000 TL | Satış temsilcisi |
| 50.000+ TL | Satış müdürü |

---

## Ödeme Entegrasyonu

### Ödeme Yöntemleri

| Yöntem | B2C | B2B |
|--------|-----|-----|
| Kredi kartı | ✓ | ✓ |
| Banka kartı | ✓ | ✓ |
| Havale/EFT | ✓ | ✓ |
| Kapıda ödeme | ✓ | ✗ |
| Cari hesap | ✗ | ✓ |
| Çek/Senet | ✗ | ✓ |

### Sanal POS Akışı

```
Kart Bilgileri
    ↓
3D Secure (SMS doğrulama)
    ↓
Banka Onayı
    ↓
Provizyon
    ↓
Sipariş Tamamlandı
```

---

## Kargo Entegrasyonu

### Kargo Seçimi

| Kargo Tipi | Süre | Maliyet |
|-----------|------|---------|
| Standart | 2-4 gün | Düşük |
| Hızlı | 1-2 gün | Orta |
| Aynı gün | Aynı gün | Yüksek |
| Mağazadan teslim | - | Ücretsiz |

### Kargo Entegrasyon Akışı

```
Sipariş Onayı
    ↓
Barkod Oluştur
    ↓
Manifest Gönder
    ↓
Kargo Teslim
    ↓
Takip Numarası Al
    ↓
Müşteriye Bildirim
    ↓
Teslimat Durumu Güncelle
```

## Bildirim Sistemi

### Email Bildirimleri

| Tetikleyici | Bildirim |
|-------------|----------|
| Sipariş oluşturuldu | Sipariş onay emaili |
| Ödeme alındı | Ödeme onay emaili |
| Kargoya verildi | Kargo bilgi emaili |
| Teslim edildi | Teslimat onay emaili |
| İade onaylandı | İade onay emaili |

### SMS Bildirimleri

```
"Siparişiniz kargoya verildi.
Takip No: 123456789
Takip: kargolink.com/123456789"
```

## İade Akışı

### İade Süreci

```
İade Talebi
    ↓
Talep İnceleme
    ↓
Onay/Red
    ↓
Kargo Kodu Gönder
    ↓
Ürün Teslim Al
    ↓
Kalite Kontrol
    ↓
Para İadesi
```

### İade Nedenleri

| Neden | Oran |
|-------|------|
| Yanlış ürün | %25 |
| Hasarlı ürün | %20 |
| Beden/renk uyumsuz | %30 |
| Beklentiyi karşılamadı | %15 |
| Vazgeçme | %10 |

## Hazır Platform vs Özel Yazılım

| Sipariş Akışı Özelliği | Hazır Platform | Özel Yazılım |
|------------------------|---------------|--------------|
| B2B onay mekanizması | Yok/Sınırlı | Tam destek |
| Özel sipariş durumları | Sınırlı | Sınırsız |
| Çoklu kargo entegrasyon | Eklenti | Native |
| Kredi limit kontrolü | Yok | Tam |
| Özel bildirim kuralları | Sınırlı | Esnek |

## Sipariş Akışı Optimizasyonu

### Checkout Optimizasyonu

- Tek sayfa checkout
- Misafir checkout seçeneği
- Adres otomatik tamamlama
- Kayıtlı kart kullanımı
- Progress indicator

### Terk Edilen Sepet

```
Sepet terk edildi
    ↓
1 saat sonra: Email hatırlatma
    ↓
24 saat sonra: İndirim teklifi
    ↓
72 saat sonra: Son hatırlatma
```
