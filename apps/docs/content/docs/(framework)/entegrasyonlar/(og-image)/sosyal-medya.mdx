### Standard Events

| Event | JavaScript |
|-------|------------|
| Ürün görüntüleme | `fbq('track', 'ViewContent')` |
| Sepete ekleme | `fbq('track', 'AddToCart')` |
| Checkout | `fbq('track', 'InitiateCheckout')` |
| Satın alma | `fbq('track', 'Purchase')` |

### Purchase Event Detay

```javascript
fbq('track', 'Purchase', {
  value: 599.99,
  currency: 'TRY',
  content_ids: ['SKU-001', 'SKU-002'],
  content_type: 'product',
  num_items: 2
});
```
