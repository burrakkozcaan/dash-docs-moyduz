## Muhasebe Entegrasyonları

E-ticaret sisteminin muhasebe ve fatura süreçleri ile entegrasyonu.

---

## E-Fatura / E-Arşiv

### Desteklenen Entegratörler

| Entegratör | Özellikler |
|------------|------------|
| **Foriba** | Yaygın kullanım, güçlü API |
| **Logo e-Fatura** | Logo ERP entegre |
| **Mikro e-Fatura** | Mikro ERP entegre |
| **QNB e-Fatura** | Banka entegreli |
| **İzibiz** | Kolay entegrasyon |

### Otomatik Fatura Akışı

```
Sipariş Tamamlandı
    ↓
Müşteri Tipi Kontrol (B2B/B2C)
    ↓
    ├── B2B: E-Fatura oluştur
    │       ↓
    │   GİB'e gönder
    │       ↓
    │   Onay bekle
    │
    └── B2C: E-Arşiv oluştur
            ↓
        PDF oluştur
            ↓
        Müşteriye gönder
```

### Fatura İçeriği

```json
{
  "invoiceType": "SATIS",
  "invoiceProfile": "TICARIFATURA",
  "customer": {
    "taxNumber": "1234567890",
    "title": "ABC Şirketi",
    "address": "İstanbul"
  },
  "items": [
    {
      "name": "Ürün Adı",
      "quantity": 2,
      "unitPrice": 100.00,
      "vatRate": 20,
      "total": 240.00
    }
  ],
  "totals": {
    "subtotal": 200.00,
    "vat": 40.00,
    "total": 240.00
  }
}
```

## Muhasebe Yazılımları

### Entegrasyon Kapsamı

| İşlem | E-Ticaret → Muhasebe |
|-------|---------------------|
| Satış | Satış faturası |
| İade | İade faturası |
| Ödeme | Tahsilat kaydı |
| Gider | Gider kaydı |

### Hesap Planı Eşleştirme

| E-Ticaret | Muhasebe Hesabı |
|-----------|-----------------|
| Kredi kartı satış | 102.01 |
| Havale satış | 102.02 |
| Kapıda ödeme | 120.01 |
| Kargo gideri | 760.01 |
| Komisyon gideri | 760.02 |

## KDV Yönetimi

### KDV Oranları

| Kategori | Oran |
|----------|------|
| Temel gıda | %1 |
| Tekstil | %10 |
| Genel ürünler | %20 |

### Otomatik KDV Hesaplama

```
Ürün Fiyatı: 100 TL (KDV dahil)

KDV %20 ise:
- KDV Matrah: 83.33 TL
- KDV Tutarı: 16.67 TL
- Toplam: 100.00 TL
```

---

## Raporlama

### Muhasebe Raporları

| Rapor | Dönem | Format |
|-------|-------|--------|
| Satış raporu | Günlük/Aylık | Excel, PDF |
| KDV raporu | Aylık | GİB format |
| Cari hesap | Anlık | Excel |
| Gider raporu | Aylık | Excel |

### Otomatik Rapor Gönderimi

- Günlük satış özeti
- Haftalık performans
- Aylık KDV dökümü

## Cari Hesap Yönetimi

### B2B Cari İşlemleri

```
Sipariş (Alacak)
    ↓
Fatura Oluştur
    ↓
Ödeme Al (Borç)
    ↓
Bakiye Güncelle
```

### Cari Mutabakat

| İşlem | Tip | Tutar |
|-------|-----|-------|
| Sipariş #1001 | Alacak | 5.000 TL |
| Ödeme #P001 | Borç | 3.000 TL |
| Sipariş #1005 | Alacak | 2.500 TL |
| **Bakiye** | | **4.500 TL** |
