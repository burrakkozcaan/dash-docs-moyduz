## Kargo Entegrasyon Rehberi

Türkiye'deki tüm büyük kargo firmaları ile API entegrasyonu.

---

## Desteklenen Kargo Firmaları

### Yurtiçi Kargo

En geniş şube ağına sahip kargo firması.

**Entegrasyon Özellikleri:**
- Otomatik barkod oluşturma
- Gerçek zamanlı takip
- Teslimat SMS bildirimi
- İade yönetimi
- Kapıda ödeme

**API Metodları:**

| Metod | Açıklama |
|-------|----------|
| `createShipment` | Gönderi oluştur |
| `getTracking` | Takip sorgula |
| `cancelShipment` | Gönderi iptal |
| `getPrice` | Fiyat sorgula |

### Aras Kargo

Ekonomik fiyat ve geniş ağ.

**Özellikler:**
- Mikro kargo (küçük paketler)
- Ekspres teslimat
- Gün içi teslimat (büyükşehirler)
- Pazar günü teslimat

### MNG Kargo

Hızlı teslimat ve güçlü teknoloji altyapısı.

**Özellikler:**
- Same day delivery
- Akşam teslimat
- Randevulu teslimat
- Webhook entegrasyonu

### Sürat Kargo

Esnek teslimat seçenekleri.

**Özellikler:**
- Kapıda ödeme (düşük komisyon)
- Kolay iade
- Şubede teslim
- Posta kutusu teslimat

## Entegrasyon Akışı

### Sipariş → Kargo

```
Sipariş Onaylandı
    ↓
Kargo Firması Seç (kurala göre)
    ↓
Gönderi Oluştur (API)
    ↓
Barkod Al
    ↓
Etiket Yazdır
    ↓
Takip Numarası Kaydet
    ↓
Müşteriye SMS/Email
```

### Otomatik Firma Seçimi

| Kural | Firma |
|-------|-------|
| Desi < 1 | Mikro Kargo (Aras) |
| İstanbul içi | MNG Express |
| Kapıda ödeme | Sürat Kargo |
| Standart | Yurtiçi Kargo |

---

## Desi Hesaplama

```
Desi = (En × Boy × Yükseklik) / 3000

Örnek:
30cm × 20cm × 15cm = 9000 / 3000 = 3 desi
```

**Ücretlendirme:**
- Ağırlık veya desi, hangisi büyükse o baz alınır
- 1 desi = 0.2 kg eşdeğer

## Teslimat Takibi

### Webhook Events

| Event | Açıklama |
|-------|----------|
| `CREATED` | Gönderi oluşturuldu |
| `PICKED_UP` | Kargoya verildi |
| `IN_TRANSIT` | Yolda |
| `OUT_FOR_DELIVERY` | Dağıtımda |
| `DELIVERED` | Teslim edildi |
| `RETURNED` | İade edildi |

### Müşteri Bildirimleri

- Kargo veriliş SMS/Email
- Dağıtımda SMS
- Teslim edildi SMS
- Teslimat başarısız bilgilendirme

## İade Yönetimi

### İade Süreci

```
İade Talebi
    ↓
Onay (manuel/otomatik)
    ↓
İade Kargo Kodu Oluştur
    ↓
Müşteriye Bildirim
    ↓
Kargo Takip
    ↓
Depo Teslim Alım
    ↓
Ürün Kontrolü
    ↓
İade İşlemi Tamamla
```

### İade Kargo Ücreti

| Senaryo | Ücret |
|---------|-------|
| Ücretsiz iade (satıcı) | Satıcı öder |
| Müşteri kaynaklı | Müşteri öder |
| Değişim | Ücretsiz |
