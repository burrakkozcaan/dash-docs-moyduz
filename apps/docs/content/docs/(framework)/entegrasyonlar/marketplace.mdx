## Marketplace Entegrasyon Rehberi

Türkiye'nin büyük marketplace platformları ile ürün ve sipariş senkronizasyonu.

---

## Desteklenen Platformlar

### Trendyol

Türkiye'nin en büyük marketplace platformu.

**Entegrasyon Kapsamı:**

| Modül | Açıklama |
|-------|----------|
| **Ürün** | Ürün oluşturma, güncelleme, silme |
| **Stok** | Gerçek zamanlı stok senkron |
| **Fiyat** | Otomatik fiyat güncelleme |
| **Sipariş** | Sipariş çekme, durum güncelleme |
| **Kargo** | Kargo entegrasyonu |

**Özellikler:**
- Kategori mapping
- Marka eşleştirme
- Varyant yönetimi
- Kampanya entegrasyonu

### Hepsiburada

Geniş müşteri kitlesine erişim.

**Entegrasyon Kapsamı:**
- Ürün listeleme
- Stok senkronizasyonu
- Sipariş yönetimi
- Fatura entegrasyonu
- İade yönetimi

### Amazon Türkiye

Global marketplace erişimi.

**Özellikler:**
- FBA (Fulfillment by Amazon) desteği
- Çoklu ülke satış
- A+ Content
- Brand Registry

### n11

Alternatif marketplace seçeneği.

**Entegrasyon Kapsamı:**
- Ürün yönetimi
- Sipariş senkron
- Kampanya entegrasyonu
- Mağaza yönetimi

---

## Entegrasyon Akışı

### Ürün Senkronizasyonu

```
E-Ticaret Ürün Oluştur/Güncelle
    ↓
Kategori Mapping Kontrol
    ↓
Marka Eşleştirme
    ↓
Görsel Yükleme
    ↓
Marketplace API Çağrısı
    ↓
Durum Kontrolü
    ↓
Hata varsa Log & Bildirim
```

### Sipariş Senkronizasyonu

```
Marketplace Sipariş Kontrolü (Periyodik)
    ↓
Yeni Sipariş Çek
    ↓
E-Ticaret Sipariş Oluştur
    ↓
Stok Düş
    ↓
Kargo Entegrasyonu
    ↓
Marketplace Durum Güncelle
```

## Fiyat Stratejileri

### Kanal Bazlı Fiyatlandırma

| Kanal | Strateji |
|-------|----------|
| **Website** | Baz fiyat |
| **Trendyol** | Baz + %5 (komisyon karşılığı) |
| **Hepsiburada** | Baz + %7 |
| **Amazon** | Baz + %10 |

### Kampanya Yönetimi

- Marketplace kampanyalarına otomatik katılım
- Fiyat alt/üst limit kontrolü
- Kampanya bazlı stok ayırma

---

## Stok Yönetimi

### Çoklu Kanal Stok

```
Toplam Stok: 100 adet
    ├── Website: 40 adet
    ├── Trendyol: 30 adet
    ├── Hepsiburada: 20 adet
    └── Rezerv: 10 adet
```

### Stok Güncelleme Kuralları

| Olay | Aksiyon |
|------|---------|
| Satış (herhangi kanal) | Tüm kanallarda stok düş |
| Stok girişi | Oransal dağıtım |
| Kritik stok | Sadece website'de sat |

---

## Hata Yönetimi

### Yaygın Hatalar

| Hata | Çözüm |
|------|-------|
| Kategori bulunamadı | Mapping tablosu güncelle |
| Marka onaysız | Marka başvurusu yap |
| Görsel hatalı | Boyut/format düzelt |
| Stok yetersiz | Stok kontrolü ekle |

### Monitoring

- Senkronizasyon logları
- Hata bildirimleri
- Performans raporları
