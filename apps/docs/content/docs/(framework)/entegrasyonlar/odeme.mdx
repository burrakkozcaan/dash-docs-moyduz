## Ödeme Sistemi Entegrasyonları

Türkiye'de yaygın kullanılan tüm ödeme sistemleri ile entegrasyon sağlanır.

---

## Sanal POS Entegrasyonları

### iyzico

Kolay entegrasyon ve marketplace desteği ile öne çıkan ödeme altyapısı.

**Özellikler:**
- Hızlı entegrasyon (sandbox ortamı)
- Marketplace ödeme dağıtımı
- Taksit seçenekleri
- Fraud koruması
- Recurring ödeme

**Komisyon:** %2.49 + 0.25 TL (standart)

### PayTR

Düşük komisyon oranları ve hızlı para transferi.

**Özellikler:**
- Aynı gün para transferi
- Düşük komisyon
- Tüm bankalar taksit
- Link ile ödeme
- iFrame entegrasyonu

**Komisyon:** %1.99 + 0.15 TL (anlaşmaya göre)

### Param (Türkiye Finans)

Kapsamlı raporlama ve güçlü API desteği.

**Özellikler:**
- Detaylı raporlama
- REST API
- Webhook desteği
- Multi-currency
- B2B ödeme

## Banka Sanal POS

Doğrudan banka entegrasyonu ile düşük komisyon.

| Banka | Özellikler |
|-------|------------|
| **İş Bankası** | Maximum taksit, puan kullanımı |
| **Garanti** | Bonus taksit, 3D Secure |
| **Yapı Kredi** | World taksit, tek çekim |
| **Akbank** | Axess taksit, kampanyalar |
| **QNB** | Paraf taksit, loyalty |

## Alternatif Ödeme Yöntemleri

### Havale/EFT

```
Sipariş Oluştur
    ↓
Havale Bilgisi Göster
    ↓
Ödeme Bildirimi (manuel/otomatik)
    ↓
Onay & Kargolama
```

**Otomatik Havale Kontrolü:**
- Banka API entegrasyonu
- Açıklama bazlı eşleştirme
- Otomatik sipariş onayı

### Kapıda Ödeme

- Nakit ödeme
- Kapıda kredi kartı (POS cihazı)
- Komisyon: %2-3 (kargo firmasına göre)

---

## 3D Secure

Tüm entegrasyonlarda 3D Secure zorunlu veya opsiyonel olarak yapılandırılabilir.

| Tip | Açıklama |
|-----|----------|
| **3D Secure** | SMS doğrulama |
| **3D Secure 2.0** | Gelişmiş risk analizi |
| **Non-3D** | Düşük tutarlı işlemler |

---

## Taksit Yönetimi

### Dinamik Taksit Tablosu

- Banka bazlı taksit seçenekleri
- Minimum tutar kontrolü
- Komisyon hesaplama
- Vade farkı ekleme

### Örnek Taksit Tablosu

| Taksit | Komisyon | 1.000 TL |
|--------|----------|----------|
| Tek Çekim | %0 | 1.000 TL |
| 3 Taksit | %3 | 1.030 TL |
| 6 Taksit | %6 | 1.060 TL |
| 9 Taksit | %9 | 1.090 TL |
