## Ürün API Dokümantasyonu

Ürün oluşturma, güncelleme, listeleme ve silme işlemleri için API referansı.

---

## Endpoint Listesi

| Metod | Endpoint | Açıklama |
|-------|----------|----------|
| GET | `/products` | Ürün listesi |
| GET | `/products/{id}` | Ürün detayı |
| POST | `/products` | Ürün oluştur |
| PUT | `/products/{id}` | Ürün güncelle |
| DELETE | `/products/{id}` | Ürün sil |
| GET | `/products/{id}/variants` | Varyantlar |
| PATCH | `/products/{id}/stock` | Stok güncelle |

---

## GET /products

Ürün listesini getirir.

### Query Parameters

| Parametre | Tip | Zorunlu | Açıklama |
|-----------|-----|---------|----------|
| `page` | integer | Hayır | Sayfa numarası (default: 1) |
| `limit` | integer | Hayır | Sayfa başına kayıt (default: 20, max: 100) |
| `category` | string | Hayır | Kategori ID filtresi |
| `minPrice` | number | Hayır | Minimum fiyat |
| `maxPrice` | number | Hayır | Maximum fiyat |
| `inStock` | boolean | Hayır | Stokta olanlar |
| `sort` | string | Hayır | Sıralama: price, -price, createdAt |

### Response

```json
{
  "success": true,
  "data": [
    {
      "id": "PROD-001",
      "sku": "SKU-12345",
      "name": "Örnek Ürün",
      "slug": "ornek-urun",
      "price": 199.99,
      "comparePrice": 249.99,
      "stock": 50,
      "category": {
        "id": "CAT-001",
        "name": "Elektronik"
      },
      "images": [
        "https://cdn.example.com/img1.jpg"
      ],
      "createdAt": "2026-01-01T00:00:00Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 150,
    "totalPages": 8
  }
}
```

## POST /products

Yeni ürün oluşturur.

### Request Body

```json
{
  "sku": "SKU-12345",
  "name": "Yeni Ürün",
  "description": "Ürün açıklaması",
  "price": 199.99,
  "comparePrice": 249.99,
  "stock": 100,
  "categoryId": "CAT-001",
  "images": [
    "https://cdn.example.com/img1.jpg"
  ],
  "attributes": {
    "renk": "Mavi",
    "beden": "M"
  },
  "seo": {
    "title": "SEO Başlık",
    "description": "SEO Açıklama"
  }
}
```

### Response

```json
{
  "success": true,
  "data": {
    "id": "PROD-002",
    "sku": "SKU-12345",
    "name": "Yeni Ürün",
    "createdAt": "2026-01-24T12:00:00Z"
  }
}
```

---

## PUT `/products/{id}`

Mevcut ürünü günceller.

### Path Parameters

| Parametre | Tip | Açıklama |
|-----------|-----|----------|
| `id` | string | Ürün ID |

### Request Body

POST ile aynı yapı. Sadece güncellenmek istenen alanlar gönderilebilir.

---

## PATCH `/products/{id}/stock`

Ürün stok miktarını günceller.

### Request Body

```json
{
  "quantity": 50,
  "operation": "set"  // set, increment, decrement
}
```

### Response

```json
{
  "success": true,
  "data": {
    "id": "PROD-001",
    "previousStock": 30,
    "newStock": 50
  }
}
```

## Hata Kodları

| Kod | Açıklama |
|-----|----------|
| `PRODUCT_NOT_FOUND` | Ürün bulunamadı |
| `SKU_ALREADY_EXISTS` | SKU zaten kullanımda |
| `INVALID_CATEGORY` | Geçersiz kategori ID |
| `STOCK_INSUFFICIENT` | Yetersiz stok |
