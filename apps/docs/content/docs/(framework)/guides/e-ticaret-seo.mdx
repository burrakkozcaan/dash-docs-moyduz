## E-Ticaret SEO Nedir?

E-ticaret SEO, online mağazanızın Google ve diğer arama motorlarında üst sıralarda yer almasını sağlayan optimizasyon çalışmalarıdır. Özel e-ticaret sitesi, Moyduz yaklaşımıyla SEO altyapısı en baştan doğru kurulur.

## Neden E-Ticaret SEO Kritik?

- **Organik trafik ücretsiz** - Reklam maliyeti yok
- **Yüksek dönüşüm** - Arama yapan kullanıcı satın almaya hazır
- **Uzun vadeli yatırım** - Bir kez yapılan çalışma yıllarca değer üretir
- **Marka güveni** - Üst sıralarda olmak güven oluşturur

<Callout type="info">
  **İstatistik:** E-ticaret trafiğinin %40'ı organik aramalardan gelir.
</Callout>

## E-Ticaret SEO Temelleri

### 1. Teknik SEO

Özel e-ticaret sitesi, Moyduz yaklaşımıyla teknik altyapı en baştan optimize edilir:

- **Site hızı** - Core Web Vitals uyumu
- **Mobile-first** - Mobil öncelikli tasarım
- **Crawlability** - Google'ın siteyi taraması
- **Indexability** - Sayfaların indekslenmesi
- **SSL** - HTTPS zorunluluğu

### 2. On-Page SEO

Her ürün ve kategori sayfası için:

- **Title tag** - Benzersiz, anahtar kelime içeren başlık
- **Meta description** - Tıklamaya teşvik eden açıklama
- **H1-H6** - Doğru başlık hiyerarşisi
- **URL yapısı** - Temiz, anlaşılır URL'ler
- **Alt text** - Görsel açıklamaları

### 3. İçerik SEO

- **Ürün açıklamaları** - Benzersiz, detaylı açıklamalar
- **Kategori içerikleri** - Kategori sayfalarında bilgilendirici metin
- **Blog** - Bilgilendirici içerikler
- **SSS** - Sık sorulan sorular

## Hazır Platformların SEO Sınırları

Hazır e-ticaret platformlarında sık karşılaşılan SEO sorunları:

| Sorun | Açıklama | Özel Yazılımda |
|-------|----------|----------------|
| URL yapısı | Değiştirilemez | Tam kontrol |
| Schema markup | Sınırlı | Özel şema |
| Sayfa hızı | Standart | Optimize |
| Canonical | Otomatik (hatalı) | Manuel kontrol |
| Robots.txt | Kısıtlı | Tam kontrol |

## E-Ticaret SEO Checklist

### Teknik
- [ ] SSL sertifikası aktif
- [ ] Site hızı 3 saniyenin altında
- [ ] Mobile-friendly test geçiyor
- [ ] XML sitemap oluşturuldu
- [ ] Robots.txt optimize edildi

### On-Page
- [ ] Her sayfada benzersiz title
- [ ] Meta description'lar yazıldı
- [ ] URL'ler SEO uyumlu
- [ ] Görsel alt text'leri eklendi
- [ ] Internal linking yapıldı

### İçerik
- [ ] Ürün açıklamaları benzersiz
- [ ] Kategori sayfalarında içerik var
- [ ] Blog aktif
- [ ] SSS sayfası mevcut
