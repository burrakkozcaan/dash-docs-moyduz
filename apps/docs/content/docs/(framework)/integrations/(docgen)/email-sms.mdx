## Email ve SMS Entegrasyonları

Müşteri iletişimi için email ve SMS entegrasyonları.

---

## Email Sistemleri

### Desteklenen Servisler

| Servis | Özellikler |
|--------|------------|
| **Mailchimp** | Pazarlama otomasyonu, segmentasyon |
| **Sendinblue** | Transactional + marketing email |
| **Amazon SES** | Yüksek hacim, düşük maliyet |
| **SendGrid** | Güçlü API, detaylı analitik |

### Email Türleri

#### Transactional (Sistem) Emailler

| Event | Email |
|-------|-------|
| Kayıt | Hoşgeldin emaili |
| Sipariş | Sipariş onayı |
| Ödeme | Ödeme onayı |
| Kargo | Kargo bilgisi |
| Teslimat | Teslimat onayı |
| İade | İade onayı |

#### Marketing Emailler

- Newsletter
- Kampanya duyuruları
- Kişiselleştirilmiş öneriler
- Sepet hatırlatma
- Win-back kampanyaları

## Email Şablonları

### Sipariş Onayı Örnek

```html
Merhaba {{customer.firstName}},

Siparişiniz alındı!

Sipariş No: {{order.number}}
Toplam: {{order.total}} TL

Ürünler:
{{#each order.items}}
- {{this.name}} x {{this.quantity}} - {{this.total}} TL
{{/each}}

Teslimat Adresi:
{{order.shippingAddress.fullAddress}}

Tahmini Teslimat: {{order.estimatedDelivery}}
```

### Değişken Sistemi

| Değişken | Açıklama |
|----------|----------|
| `{{customer.firstName}}` | Müşteri adı |
| `{{order.number}}` | Sipariş numarası |
| `{{order.total}}` | Sipariş toplamı |
| `{{tracking.url}}` | Kargo takip linki |

## SMS Entegrasyonları

### Desteklenen Servisler

| Servis | Özellikler |
|--------|------------|
| **İletimerkezi** | Türkiye altyapısı |
| **JetSMS** | Toplu SMS |
| **Turkcell Mesaj** | Operatör SMS |
| **Twilio** | Global, API güçlü |

### SMS Senaryoları

| Senaryo | Mesaj |
|---------|-------|
| Sipariş | "Siparişiniz alındı. Sipariş No: #1234" |
| Kargo | "Siparişiniz kargoya verildi. Takip: YK123..." |
| Teslimat | "Siparişiniz teslim edildi." |
| OTP | "Doğrulama kodunuz: 123456" |

### SMS Şablon Örnek

```
[MARKA] Siparişiniz kargoya verildi.
Takip: {{tracking.number}}
Link: {{tracking.url}}
```

## Automation Flows

### Hoşgeldin Serisi

```
Kayıt
    ↓ (hemen)
Email 1: Hoşgeldin + İndirim kodu
    ↓ (3 gün)
Email 2: Popüler ürünler
    ↓ (7 gün)
Email 3: Kategorileri keşfet
```

### Sepet Terk

```
Sepet terk edildi
    ↓ (1 saat)
Email 1: Sepetini unuttun mu?
    ↓ (24 saat)
Email 2: %10 indirim fırsatı
    ↓ (48 saat)
SMS: Son şans!
```

### Post-Purchase

```
Teslimat sonrası
    ↓ (3 gün)
Email: Memnuniyet anketi
    ↓ (7 gün)
Email: Ürün değerlendirmesi iste
    ↓ (30 gün)
Email: İlgili ürün önerileri
```

## Analitik

### Email Metrikleri

| Metrik | Açıklama | Hedef |
|--------|----------|-------|
| Open Rate | Açılma oranı | > 20% |
| Click Rate | Tıklama oranı | > 3% |
| Bounce Rate | Geri dönme | < 2% |
| Unsubscribe | Abonelik iptal | < 0.5% |

### SMS Metrikleri

| Metrik | Açıklama |
|--------|----------|
| Delivery Rate | Teslim oranı |
| Response Rate | Yanıt oranı |
| Opt-out Rate | Çıkış oranı |
