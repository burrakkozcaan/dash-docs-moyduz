## ERP Entegrasyon Rehberi

E-ticaret sisteminizi mevcut ERP yazılımınızla entegre edin.

---

## Desteklenen ERP Sistemleri

### Logo Tiger/Wolf

Türkiye'nin en yaygın ERP sistemi ile tam entegrasyon.

**Senkronize Edilen Veriler:**

| Modül | E-Ticaret → ERP | ERP → E-Ticaret |
|-------|-----------------|-----------------|
| **Ürün** | - | Ürün bilgisi, fiyat |
| **Stok** | - | Stok miktarı |
| **Sipariş** | Sipariş oluşturma | Sipariş durumu |
| **Cari** | Yeni müşteri | Cari bakiye |
| **Fatura** | - | Fatura bilgisi |

**Entegrasyon Yöntemleri:**
- Logo REST API
- Logo Object API
- Veritabanı entegrasyonu (SQL)
- Ara katman (middleware)

### Mikro

Küçük-orta ölçekli işletmeler için ekonomik çözüm.

**Entegrasyon Kapsamı:**
- Stok kartı senkronizasyonu
- Sipariş aktarımı
- İrsaliye oluşturma
- Fatura entegrasyonu
- Cari hesap

### Netsis

Kurumsal düzey ERP entegrasyonu.

**Özellikler:**
- Web servis entegrasyonu
- Gerçek zamanlı stok
- Çoklu depo desteği
- Üretim planlama entegrasyonu

### SAP

Enterprise düzey entegrasyon.

**SAP Modülleri:**
- SAP MM (Malzeme Yönetimi)
- SAP SD (Satış ve Dağıtım)
- SAP FI (Finans)

---

## Entegrasyon Mimarisi

### Gerçek Zamanlı (Real-time)

```
E-Ticaret ←→ API Gateway ←→ ERP
     ↓
  Webhook
```

**Kullanım:** Stok kontrolü, cari sorgulama

### Periyodik (Batch)

```
E-Ticaret → Queue → Scheduler → ERP
                        ↓
                    Her 5 dk
```

**Kullanım:** Sipariş aktarımı, toplu güncelleme

### Hibrit

Kritik işlemler gerçek zamanlı, diğerleri periyodik.

## Veri Eşleştirme

### Ürün Eşleştirme

| E-Ticaret | ERP | Eşleştirme |
|-----------|-----|------------|
| SKU | Stok Kodu | Birebir |
| Barkod | GTIN | Birebir |
| Kategori | Grup Kodu | Mapping tablosu |
| Fiyat | Liste Fiyatı | Fiyat tipi seçimi |

### Sipariş Akışı

```
E-Ticaret Sipariş
    ↓
Cari Kontrolü (varsa eşle, yoksa oluştur)
    ↓
Stok Kontrolü
    ↓
ERP Sipariş Oluştur
    ↓
İrsaliye Oluştur
    ↓
Kargo Entegrasyonu
    ↓
Fatura Oluştur
```

- Manuel müdahale için bildirim

| Durum | Çözüm |
|-------|-------|
