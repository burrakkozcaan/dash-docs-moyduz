## Sipariş API Dokümantasyonu

Sipariş oluşturma, sorgulama ve yönetim işlemleri için API referansı.

---

## Endpoint Listesi

| Metod | Endpoint | Açıklama |
|-------|----------|----------|
| GET | `/orders` | Sipariş listesi |
| GET | `/orders/{id}` | Sipariş detayı |
| POST | `/orders` | Sipariş oluştur |
| PATCH | `/orders/{id}/status` | Durum güncelle |
| POST | `/orders/{id}/cancel` | Sipariş iptal |
| POST | `/orders/{id}/refund` | İade işlemi |

---

## GET /orders

Sipariş listesini getirir.

### Query Parameters

| Parametre | Tip | Zorunlu | Açıklama |
|-----------|-----|---------|----------|
| `page` | integer | Hayır | Sayfa numarası |
| `limit` | integer | Hayır | Sayfa başına kayıt |
| `status` | string | Hayır | Durum filtresi |
| `startDate` | date | Hayır | Başlangıç tarihi |
| `endDate` | date | Hayır | Bitiş tarihi |
| `customerId` | string | Hayır | Müşteri ID |

### Sipariş Durumları

| Durum | Açıklama |
|-------|----------|
| `pending` | Beklemede |
| `paid` | Ödendi |
| `processing` | Hazırlanıyor |
| `shipped` | Kargoda |
| `delivered` | Teslim Edildi |
| `cancelled` | İptal |
| `refunded` | İade Edildi |

      },
      "items": [
        {
          "productId": "PROD-001",
          "name": "Örnek Ürün",
          "quantity": 2,
          "price": 199.99,

## POST /orders

Yeni sipariş oluşturur.

### Request Body

```json
{
  "customer": {
    "email": "customer@example.com",
    "phone": "5551234567",
    "firstName": "Ahmet",
    "lastName": "Yılmaz"
  },
  "items": [
    {
      "productId": "PROD-001",
      "variantId": "VAR-001",
      "quantity": 2
    }
  ],
  "shippingAddress": {
    "firstName": "Ahmet",
    "lastName": "Yılmaz",
    "phone": "5551234567",
    "city": "İstanbul",
    "district": "Kadıköy",
    "neighborhood": "Moda",
    "address": "Moda Cad. No:1 D:5",
    "postalCode": "34710"
  },
  "billingAddress": {
    "sameAsShipping": true
  },
  "shippingMethod": "standard",
  "paymentMethod": "credit_card",

    "orderNumber": "2026012400002",
    "status": "pending",
    "paymentUrl": "https://pay.example.com/checkout/abc123",

  }

```

## PATCH `/orders/{id}/status`

Sipariş durumunu günceller.

### Request Body

```json
{
  "status": "shipped",
  "trackingNumber": "YK1234567890",
  "trackingUrl": "https://kargotakip.com/YK1234567890",
  "shippingCompany": "yurtici"
}
```

### Durum Geçişleri

```
pending → paid → processing → shipped → delivered
    ↓         ↓           ↓
cancelled  cancelled   cancelled

delivered → refunded (kısmi veya tam)
```

---

## POST `/orders/{id}/cancel`

Siparişi iptal eder.

### Request Body

```json
{
  "reason": "customer_request",
  "notes": "Müşteri vazgeçti"
}
```

### İptal Nedenleri

| Kod | Açıklama |
|-----|----------|
| `customer_request` | Müşteri talebi |
| `out_of_stock` | Stok yok |
| `payment_failed` | Ödeme başarısız |
| `fraud` | Şüpheli işlem |
| `other` | Diğer |

---

## POST `/orders/{id}/refund`

İade işlemi başlatır.

### Request Body

```json
{
  "type": "partial",
  "items": [
    {
      "orderItemId": "ITEM-001",
      "quantity": 1,
      "reason": "defective"
    }
  ],
  "refundShipping": false,
  "notes": "Ürün hasarlı geldi"
}
```

### Response

```json
{
  "success": true,
  "data": {
    "refundId": "REF-001",
    "orderId": "ORD-12345",
    "amount": 199.99,
    "status": "processing"
  }
}
```

---

## Hata Kodları

| Kod | Açıklama |
|-----|----------|
| `ORDER_NOT_FOUND` | Sipariş bulunamadı |
| `INVALID_STATUS_TRANSITION` | Geçersiz durum geçişi |
| `ALREADY_CANCELLED` | Zaten iptal edilmiş |
| `REFUND_LIMIT_EXCEEDED` | İade limiti aşıldı |
