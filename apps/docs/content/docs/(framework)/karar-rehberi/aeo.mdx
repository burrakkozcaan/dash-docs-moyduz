## AEO Nedir?

AEO (Answer Engine Optimization), sesli asistanlar ve direkt cevap veren arama motorları için optimizasyondur. Google Assistant, Siri, Alexa ve Featured Snippets'ta görünmenizi hedefler.

## Neden AEO Önemli? (Veriler)
 
Sesli ve asistif aramaların yükselişi:
 
> **Sesli Arama:** ComScore verilerine göre, 2024 yılı itibariyle tüm mobil aramaların **%50'si** sesli komutlarla yapılıyor.
 
> **Sıfırıncı Sıra:** Google aramalarının **%30'undan fazlası**, kullanıcı bir siteye tıklamadan (Zero-click searches) doğrudan sonuç sayfasında cevaplanıyor. Bu alanda ("Sıfırıncı Sıra") yer almak, marka bilinirliği için en değerli yatırımdır.

---

## Sesli Arama Davranışı

| Özellik | Metin Arama | Sesli Arama |
|---------|------------|-------------|
| Sorgu uzunluğu | 2-3 kelime | 5-7 kelime |
| Sorgu tipi | Anahtar kelime | Doğal soru |
| Beklenti | Sonuç listesi | Tek cevap |

### Örnekler

```
Metin: "iphone 15 fiyat"
Sesli: "iPhone 15'in fiyatı ne kadar?"

Metin: "en iyi kulaklık"
Sesli: "Gürültü engelleyen en iyi kulaklık hangisi?"
```

## AEO İçerik Stratejisi

### 1. FAQ Sayfaları

```markdown
### Kargo ne kadar sürer?
İstanbul içi 1-2 iş günü, şehir dışı 2-4 iş günü.

### Kargo ücreti ne kadar?
500 TL üzeri siparişlerde ücretsiz.
```

### 2. Direkt Cevap Formatı

```markdown
❌ Zayıf:
"iPhone 15 Pro, Apple'ın 2023 yılında çıkardığı..."

✓ Güçlü:
"iPhone 15 Pro 48 megapiksel ana kameraya sahiptir."
```

## Teknik Gereksinimler

### FAQ Schema

```json
{
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "Kargo ücretsiz mi?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "500 TL üzeri ücretsiz."
    }
  }]
}
```

### Speakable Schema

```json
{
  "@type": "WebPage",
  "speakable": {
    "cssSelector": [".faq-answer"]
  }
}
```
