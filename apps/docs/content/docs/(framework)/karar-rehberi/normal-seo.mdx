## Normal SEO Nedir?

Normal SEO (Geleneksel SEO), arama motorlarında organik sıralama elde etmek için yapılan klasik optimizasyon çalışmalarıdır. Google, Yandex ve Bing gibi arama motorlarında üst sıralarda yer almayı hedefler.

<Callout type="info">
  **Önemli:** Normal SEO, e-ticarette en temel trafik kaynağıdır. Diğer SEO türleri (Programmatic, GEO, AEO) normal SEO üzerine inşa edilir.
</Callout>

---

## Normal SEO Bileşenleri

### 1. Teknik SEO

Sitenizin teknik altyapısının arama motorlarına uygunluğu:

| Alan | Açıklama | Etki |
|------|----------|------|
| Site hızı | Core Web Vitals uyumu | Kritik |
| Mobile-first | Mobil öncelikli tasarım | Kritik |
| Crawlability | Bot tarama kolaylığı | Yüksek |
| Indexability | Sayfa indeksleme | Yüksek |
| SSL | HTTPS güvenliği | Orta |

**Özel e-ticaret sitesinde teknik SEO avantajı:**
- Kod üzerinde tam kontrol
- Özel sitemap yapıları
- Dinamik robots.txt
- Structured data esnekliği

### 2. On-Page SEO

Her sayfanın bireysel optimizasyonu:

- **Title Tag** - Benzersiz, anahtar kelime içeren başlık (50-60 karakter)
- **Meta Description** - Tıklamaya teşvik eden açıklama (150-160 karakter)
- **H1-H6 Hiyerarşisi** - Doğru başlık yapısı
- **URL Yapısı** - Temiz, anlaşılır URL'ler
- **İç Linkler** - Stratejik internal linking
- **Görsel Optimizasyonu** - Alt text, boyut, format

### 3. Off-Page SEO

Site dışı sinyaller ve otorite:

- **Backlink** - Kaliteli geri bağlantılar
- **Brand Mentions** - Marka bahisleri
- **Social Signals** - Sosyal medya etkileşimi
- **Local SEO** - Yerel arama optimizasyonu

## E-Ticarette Normal SEO Stratejisi

### Ürün Sayfaları

```
✓ Benzersiz ürün açıklaması (min. 300 kelime)
✓ Ürün görselleri optimize
✓ Schema markup (Product)
✓ Müşteri yorumları
✓ FAQ bölümü
```

### Kategori Sayfaları

```
✓ Kategori açıklaması (min. 500 kelime)
✓ Filtreleme SEO-friendly
✓ Breadcrumb yapısı
✓ Pagination doğru kurulum
```

### Blog / İçerik

```
✓ Bilgilendirici içerikler
✓ Ürünle ilişkili rehberler
✓ Seasonal (mevsimsel) içerikler
✓ Evergreen (her zaman güncel) içerikler
```

## Hazır Platform vs Özel Yazılım

| SEO Özelliği | Hazır Platform | Özel Yazılım |
|--------------|---------------|--------------|
| URL yapısı | Kısıtlı | Tam kontrol |
| Schema markup | Standart | Özel şema |
| Canonical | Otomatik (hatalı olabilir) | Manuel kontrol |
| Sitemap | Standart | Dinamik |
| Page Speed | Tema bağımlı | Optimize |
| JavaScript | Render sorunu | SSR/SSG |

## Normal SEO Checklist

### Teknik
- [ ] Core Web Vitals yeşil
- [ ] Mobile-friendly test geçiyor
- [ ] SSL aktif (HTTPS)
- [ ] XML Sitemap gönderildi
- [ ] Robots.txt optimize

### On-Page
- [ ] Her sayfada benzersiz title
- [ ] Meta description yazıldı
- [ ] H1 yapısı doğru
- [ ] URL'ler SEO-friendly
- [ ] Görsel alt text'ler eklendi

### İçerik
- [ ] Ürün açıklamaları benzersiz
- [ ] Kategori sayfalarında içerik var
- [ ] Blog aktif
- [ ] FAQ sayfası mevcut
