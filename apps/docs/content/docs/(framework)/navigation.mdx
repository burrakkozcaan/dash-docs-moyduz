## Ödeme Sistemleri

| Sistem | Özellikler |
|--------|-----------|
| **iyzico** | Kolay entegrasyon, marketplace desteği |
| **PayTR** | Düşük komisyon, hızlı para transferi |
| **Param** | Kapsamlı raporlama, API desteği |
| **Banka POS** | Tüm bankalar, taksit seçenekleri |

## Kargo Entegrasyonları

| Firma | Özellikler |
|-------|-----------|
| **Yurtiçi Kargo** | Geniş ağ, otomatik barkod |
| **Aras Kargo** | Ekonomik, mikro kargo |
| **MNG Kargo** | Hızlı teslimat, takip API |
| **Sürat Kargo** | Kapıda ödeme, kolay iade |

## ERP Entegrasyonları

Moyduz sistemleri, Türkiye'de yaygın kullanılan tüm ERP sistemleriyle entegre çalışır:

- **Logo Tiger/Wolf** - Cari, stok, sipariş senkronizasyonu
- **Mikro** - Fatura, irsaliye, cari entegrasyonu
- **Netsis** - Kurumsal çözümler
- **SAP** - Enterprise entegrasyon
- **Özel ERP** - API tabanlı özel entegrasyon

| **Hepsiburada** | Otomatik fiyat güncelleme |

## Hazır Platformlarda Entegrasyon Sorunu

Hazır e-ticaret platformlarında entegrasyon:

- Marketplace eklentilerine bağımlılık
- Sınırlı API erişimi
- Özel ERP desteği yok
- Ek ücretler

**Moyduz ile:** Her entegrasyon özel geliştirilir, tam kontrol sizde.
