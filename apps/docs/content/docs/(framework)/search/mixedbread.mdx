## Otomotiv Yedek Parça Projesi

Araç uyumluluk sistemi ve teknik katalog yönetimi ile yedek parça e-ticareti.

---

## Sektörel Özellikler

### Araç Uyumluluk

- **Araç Seçici** - Marka > Model > Yıl > Motor filtresi
- **OEM Kodları** - Orijinal parça kodları ile arama
- **Çapraz Referans** - Muadil parça eşleştirme
- **Uyumluluk Tablosu** - Hangi araca uyar bilgisi

### Teknik Katalog

- **Teknik Şema** - Parça şemaları ve diyagramlar
- **Montaj Rehberi** - Kurulum talimatları
- **Teknik Özellikler** - Detaylı ürün bilgileri
- **Video Rehber** - Montaj videoları

---

## Arama Özellikleri

### Araç ile Arama

```
1. Marka Seç (Ford, Volkswagen, BMW...)
2. Model Seç (Focus, Golf, 3 Serisi...)
3. Yıl Seç (2020, 2021, 2022...)
4. Motor Seç (1.6 TDI, 2.0 TFSI...)
5. Uyumlu Parçalar Listele
```

### OEM Kodu ile Arama

| Kod Tipi | Örnek |
|----------|-------|
| OEM | 1K0615301AA |
| Aftermarket | BPW-301-AA |
| Barkod | 8691234567890 |
