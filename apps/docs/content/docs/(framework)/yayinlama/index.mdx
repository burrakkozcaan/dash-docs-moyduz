## E-Ticaret Maliyetleri

E-ticaret projesinin maliyeti, seçilen model ve ihtiyaçlara göre değişir.

<Callout type="info">
  **Önemli:** Maliyet hesabında sadece başlangıç değil, 3 yıllık TCO (Total Cost of Ownership) değerlendirilmelidir.
</Callout>

---

## Maliyet Karşılaştırması

### Hazır E-Ticaret (SaaS)

| Kalem | Aylık Maliyet |
|-------|---------------|
| Platform ücreti | 500 - 3.000 TL |
| Eklentiler | 200 - 1.500 TL |
| Transaction fee | %1-3 |
| Tema | 1.000 - 5.000 TL (tek seferlik) |
| **Toplam/ay** | 1.000 - 5.000+ TL |

### Özel E-Ticaret (Moyduz)

| Kalem | Maliyet |
|-------|---------|
| Proje geliştirme | Proje bazlı |
| Hosting | 500 - 2.000 TL/ay |
| Bakım | Opsiyonel |
| Transaction fee | Yok |
| **Toplam/ay** | 500 - 2.000 TL |

---

## 3 Yıllık TCO Analizi

| Senaryo | Hazır Platform | Özel Yazılım |
|---------|---------------|--------------|
| Yıl 1 | 24.000 TL | Proje + 12.000 TL |
| Yıl 2 | 36.000 TL | 12.000 TL |
| Yıl 3 | 48.000 TL | 12.000 TL |
| **3 Yıl Toplam** | 108.000+ TL | Proje + 36.000 TL |

<Callout type="warn">
  **Not:** Hazır platformlarda büyüdükçe plan yükseltme zorunluluğu maliyeti artırır.
</Callout>

---

## Maliyet Faktörleri

### Başlangıç Maliyetleri

- **Tasarım** - UI/UX tasarım çalışması
- **Geliştirme** - Yazılım geliştirme
- **Entegrasyonlar** - Ödeme, kargo, ERP
- **İçerik** - Ürün girişi, görsel

### Operasyonel Maliyetler

- **Hosting** - Sunucu barındırma
- **SSL** - Güvenlik sertifikası
- **CDN** - İçerik dağıtım
- **Bakım** - Güncelleme, destek

### Gizli Maliyetler

Hazır platformlarda dikkat edilmesi gerekenler:
- Transaction fee (satış komisyonu)
- Eklenti ücretleri
- Limit aşım ücretleri
- Premium destek

---

## ROI Hesaplama

### Örnek Senaryo

- Aylık 100.000 TL ciro
- %2 transaction fee = 2.000 TL/ay
- Yıllık transaction fee = 24.000 TL

**Özel yazılımda:** Transaction fee yok → 24.000 TL/yıl tasarruf

---

## Hangi Model Daha Uygun?

### Hazır Platform Uygun:
- Başlangıç bütçesi < 50.000 TL
- Yıllık ciro < 500.000 TL
- Standart B2C satış
- Teknik ekip yok

### Özel Yazılım Uygun:
- Yıllık ciro > 1.000.000 TL
- B2B satış planı
- Özel entegrasyon ihtiyacı
- Uzun vadeli yatırım
