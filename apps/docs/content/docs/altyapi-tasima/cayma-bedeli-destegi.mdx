# Cayma Bedeli Desteği: Eski Altyapı Cezanızı Biz Ödüyoruz

Mevcut e-ticaret platformunuzda **mahsur mu kaldınız**? Sözleşme bitmeden çıkmak için **cayma bedeli** mi isteniyor?

**Korkmayın.** Cayma bedelinizi biz ödüyoruz. Üstelik SEO trafiğinizi kaybetmeden, sıfır veri kaybıyla geçiş yapıyoruz.

## Neden Bu Desteği Veriyoruz?

Çünkü biliyoruz ki:

- ❌ **Büyüdükçe cezalandırılıyorsunuz** (trafik limiti, paket yükseltme)
- ❌ **Sürpriz enflasyon zamları** ile karşı karşıyasınız
- ❌ **Kod sahipliğiniz yok**, kiracı konumundasınız
- ❌ **Verileriniz rehin**, kolayca çıkamıyorsunuz

**Biz farklıyız.** Size **özgürlük** satıyoruz.

## Cayma Bedeli Nasıl Çalışır?

### Adım 1: Mevcut Sözleşmenizi İnceleyin

```
Sözleşmenizde şu maddeleri arayın:
├── "Cayma bedeli"
├── "Erken iptal"
├── "Kalan ayların %X'i"
├── "Setup ücreti iade edilmez"
└── "Veri export ücreti"
```

### Adım 2: Belgeleyin

```
Gerekli belgeler:
├── Sözleşme fotokopisi
├── Cayma bedeli faturası/resmi yazı
├── Son 3 ay ödeme dekontu
└── Kapanış tarihi
```

### Adım 3: Mahsuplaşma

```
Örnek Senaryo:
Mevcut cayma bedeli:    ₺25.000
Moyduz Ticaret Paketi:  ₺350.000
────────────────────────────
Ödenecek:              ₺325.000
```

**Cayma bedeliniz, paket fiyatından düşülür.**

## SEO Kaybı Olmadan Geçiş

### 301 Yönlendirme

```
Eski URL                    → Yeni URL
/ikas/urun/urun-adi.html    → /urun/urun-adi
/ideasoft/kategori/         → /kategori/
/ticimax/sayfa/             → /sayfa/
```

**Her eski URL, yeni URL'e yönlendirilir.**

### Search Console

```
1. Yeni sitemap.xml gönder
2. Adres değişikliği bildir
3. 404 hatalarını takip et
4. Index durumunu izle
```

### Trafik Koruma

```
Beklenen: %5-10 geçici düşüş (2-4 hafta)
Sonra:    %100 geri dönüş (4-8 hafta)
Uzun vade: %20-50 artış (SEO optimizasyonu ile)
```

## Gerçek Örnekler

### Örnek 1: Tekstil Markası (İkas'tan Geçiş)

**Durum:**
- Aylık ciro: ₺400.000
- İkas cayma: ₺18.000
- Trafik: 30K/gün

**Sonuç:**
```
Cayma bedeli:     ₺18.000 (Moyduz ödedi)
Geçiş süresi:     3 hafta
SEO kaybı:        %8 (2 hafta, geri döndü)
3. ay ciro:       ₺520.000 (+%30)
```

### Örnek 2: Elektronik Satıcısı (IdeaSoft'tan)

**Durum:**
- Aylık ciro: ₺1.200.000
- IdeaSoft cayma: ₺45.000
- 50K ürün, 200K müşteri

**Sonuç:**
```
Cayma bedeli:     ₺45.000 (Moyduz ödedi)
Geçiş süresi:     4 hafta
Veri kaybı:       0 ürün, 0 müşteri
Tahsilat vadesi:  35 gün → 1 gün
Nakit akışı:      +₺800.000
```

### Örnek 3: B2B Toptancı (Ticimax'tan)

**Durum:**
- B2B modülü yetersiz
- Ticimax iptal: ₺70.000
- Özel fiyatlandırma gerekli

**Sonuç:**
```
Cayma bedeli:     ₺50.000 (kısmi destek)
B2B modülü:       Özel geliştirme
Bayii portalı:    2 haftada hazır
Sipariş artışı:   +%45 (3 ayda)
```

## Sık Sorulan Sorular

<Accordions>

<Accordion title="Cayma bedeli mutlaka ödenmeli mi?">

**Hayır.** Pazarlık yapabilirsiniz:

1. "Ödeyecek gücüm yok, yapılandırma yapın"
2. "Peşin ödersem indirim olur mu?"
3. "Başka firmaya devretsem?"

Ama en temiz yol: **Cayma bedelini ödeyip çıkmak**.

Biz de bu yüzden **destek veriyoruz**.

</Accordion>

<Accordion title="Verilerimi export edebilir miyim?">

**Evet, yasal hakkınız.**

KVKK ve Ticaret Kanunu gereği:
- Ürün verileri (CSV/XML)
- Müşteri verileri (CSV)
- Sipariş geçmişi (CSV/XML)

**Export ücreti** talep edilemez (etmeleri yasal değil).

</Accordion>

<Accordion title="Geçiş sırasında satışlar durur mu?">

**Hayır.**

Paralel koşma yapıyoruz:
1. Eski site çalışır
2. Yeni site hazırlanır
3. Test edilir
4. DNS switch (5 dakika)
5. Eski site kapatılır

**Downtime: Maksimum 5 dakika**

</Accordion>

<Accordion title="SEO sıralamam düşer mi?">

**Geçici düşüş normal** (%5-15, 2-4 hafta).

Ama doğru yapılırsa:
- 301 yönlendirmeler
- URL yapısı koruma
- Meta tag'ler aynı
- Sitemap yenileme

**4-8 haftada %100 geri döner.**

Uzun vadede **artar** (daha iyi altyapı).

</Accordion>

</Accordions>
