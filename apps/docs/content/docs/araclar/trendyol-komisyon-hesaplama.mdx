# Trendyol Komisyon Hesaplama

Trendyol'da satış yapmanın **gerçek maliyeti** nedir? Sadece komisyon değil, **kargo, iade, reklam** ve diğer gizli maliyetleri de hesaplayın.

## Gizli Maliyetler

### 1. Kargo Maliyeti

```
Alıcı ödemeli:  ₺35-₺55 (ürün başı)
Ücretsiz kargo: Satıcı karşılar (₺35-₺55)
```

### 2. Hizmet Bedeli

```
Sabit ücret: ₺2.5-₺5/sipariş
```

### 3. İade Maliyeti

```
İade oranı: %5-15 (kategoriye göre)
İade kargo: Satıcı karşılar (₺35-₺55)
Ürün değeri kaydı: %100 kayıp
```

### 4. Reklam Maliyeti

```
Zorunlu reklam: Cironun %5-10%
Sponsorlu ürün: Tıklama başı ₺0.5-₺3
Flash deal katılım: ₺5.000-₺50.000/gün
```

### 5. Ödeme Vadeleri

```
Tahsilat: 30-45 gün
Ertesi gün tahsilat: +%2 komisyon
Nakit akışı kaybı: %5-10 (enflasyon)
```

## Hesaplama Formülü

```
Toplam Maliyet = 
  Komisyon 
  + Kargo 
  + Hizmet Bedeli 
  + İade Maliyeti 
  + Reklam 
  + Ödeme Vadesi Kaybı
```

## Örnek Hesaplama

### Senaryo: Moda Satıcısı

```
Ürün fiyatı:      ₺500
Aylık satış:      500 adet
Aylık ciro:       ₺250.000

Komisyon (%18):   ₺45.000
Kargo (₺40 x 500): ₺20.000
Hizmet bedeli:    ₺1.250 (₺2.5 x 500)
İade (%10):       ₺2.500 (50 ürün x ₺50)
Reklam (%7):      ₺17.500
Vade kaybı (%5):  ₺12.500
────────────────────────────
TOPLAM MALİYET:   ₺98.750

NET KÂR:          ₺151.250
KÂR MARJI:        %60.5 (brüt %100 ise)
```

## Ne Zaman Kendi Siteniz?

### Trendyol'da Kalın Eğer:

- ✅ Yeni başlıyorsunuz
- ✅ Pazar yeri trafiği gerekli
- ✅ Marka bilinirliği yok
- ✅ Aylık ciro < ₺100K

### Kendi Sitenize Geçin Eğer:

- ✅ Aylık ciro > ₺200K
- ✅ Tekrarlayan müşteri var
- ✅ Marka bilinirliği oluştu
- ✅ Nakit akışı önemli
