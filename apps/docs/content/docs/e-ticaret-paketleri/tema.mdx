# E-ticaret Temaları

Müşterilerinizi etkilemek ve satışlarınızı artırmak için **sektörünüze özel** tasarlanmış **hazır e-ticaret temalarını** keşfedin.

## Tema Özellikleri

### %100 Mobil Uyumlu (Responsive)
Tüm temalarımız mobil cihazlar, tabletler ve masaüstü bilgisayarlarda kusursuz görünür. Google'ın mobil öncelikli indeksleme kriterlerine tam uyumludur.

### Dönüşüm Odaklı Tasarım
Sepete ekleme oranlarını artıran, kullanıcı deneyimi (UX) testlerinden geçmiş modern arayüzler.

### Sektörel Çözümler
*   **Moda ve Giyim:** Görsel odaklı, şık sliderlar.
*   **Elektronik:** Detaylı filtreleme ve karşılaştırma özellikleri.
*   **Gıda ve Market:** Hızlı sepete ekleme ve tazelik vurgusu.

## Kolay Tasarım Yönetimi

Kod bilgisi gerektirmeden renkleri ve fontları panelden değiştirebilir veya CSS ile detaylı düzenlemeler yapabilirsiniz.

```css title="global.css"
@theme {
  --color-brand: hsl(var(--primary)); /* Marka renginizi buraya girin */
  --font-heading: 'Inter', sans-serif;
}
```
