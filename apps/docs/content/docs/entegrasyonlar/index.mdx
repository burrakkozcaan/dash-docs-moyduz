# Entegrasyonlar

Moyduz, **açık kaynak kod yapısı** sayesinde istediğiniz sisteme entegre olur. Hazır platformların aksine, marketplace onayı beklemenize gerek yok.

## Entegrasyon Mimarisi

Tüm entegrasyonlar **RESTful API** veya **webhook** tabanlı çalışır:

```
Moyduz Backend
    ↕ REST API / Webhook
Ödeme Sistemi (iyzico, PayTR...)
    ↕
Kargo Sistemi (Yurtiçi, Aras...)
    ↕
ERP (Logo, Mikro...)
    ↕
Marketplace (Trendyol, Hepsiburada...)
```
