## Moyduz Anti-SaaS Manifestosu

> **"Kiralama Dönemi Bitti"**

1. **Sermayeniz Size Aittir** - Yazılım şirketine değil
2. **Büyümeniz Cezalandırılamaz** - Limitsiz ölçeklenme
3. **Enflasyon Kötü Hizmetin Bahanesi Olamaz** - Şeffaf fiyat
4. **Verileriniz ve Kodlarınız Rehin Alınamaz** - Tam sahiplik
5. **30 Bin Kişinin Kullandığı Şablona Marka Denmez** - Özel UI/UX
