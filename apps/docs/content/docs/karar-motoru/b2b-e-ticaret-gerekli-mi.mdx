# B2B E-Ticaret Gerekli mi?

B2B (Business-to-Business) e-ticaret, işletmeler arası satışları dijitalleştirir. Ama **her işletme için gerekli değil**.

Bu rehber, B2B e-ticaretin **sizin için gerekli olup olmadığına** karar vermenize yardımcı olur.

## B2B Nedir?

**B2B (Business-to-Business):** İşletmelerin diğer işletmelere satış yapması.

**Örnekler:**
- Toptancı → Perakendeci
- Üretici → Dağıtıcı
- Marka → Bayii

## B2B İhtiyaç Checklist

Eğer aşağıdaki sorulardan **3 veya daha fazlasına "Evet"** diyorsanız, B2B e-ticaret sizin için gereklidir:

### İş Modeli

- [ ] **Bayii veya distribütör ağınız var mı?**
- [ ] **Toptan satış yapıyor musunuz?**
- [ ] **Kurumsal müşterileriniz var mı?**
- [ ] **Tekrarlayan siparişler alıyor musunuz?**

### Fiyatlandırma

- [ ] **Müşteriye özel fiyatlandırma yapıyor musunuz?**
- [ ] **Miktar bazlı indirimler veriyor musunuz?**
- [ ] **Farklı müşteri grupları için farklı fiyatlar var mı?**

### Ödeme ve Finans

- [ ] **Vadeli satış yapıyor musunuz?**
- [ ] **Kredi limiti yönetimi yapıyor musunuz?**
- [ ] **Fatura ile ödeme kabul ediyor musunuz?**

### Operasyon

- [ ] **ERP sisteminizle entegrasyon kritik mi?**
- [ ] **Stok takibi karmaşık mı? (1000+ ürün)**
- [ ] **Sipariş onay süreçleri var mı?**

## B2B Özellikleri

### Temel B2B Fonksiyonları

```
1. Müşteri Grupları
   - Bayii
   - Distribütör
   - Kurumsal
   - Perakende

2. Fiyatlandırma
   - Grup bazlı fiyat
   - Miktar indirimleri
   - Özel fiyat listeleri
   - Sözleşme bazlı fiyat

3. Ödeme
   - Vadeli ödeme (30-60-90 gün)
   - Kredi limiti
   - Fatura ile ödeme
   - Cari hesap takibi

4. Sipariş
   - Toplu sipariş
   - Tekrarlayan sipariş
   - Onay akışları
   - Minimum sipariş miktarı
```

## B2B vs B2C Karşılaştırması

| Özellik | B2C | B2B |
|---------|-----|-----|
| **Müşteri** | Son kullanıcı | İşletme |
| **Sipariş Değeri** | Düşük | Yüksek |
| **Karar Süreci** | Hızlı | Uzun |
| **Ödeme** | Peşin/Kredi kartı | Vadeli/Fatura |
| **Fiyat** | Sabit | Müşteri bazlı |
| **İlişki** | Kısa vadeli | Uzun vadeli |

## B2B Ne Zaman Gerekli?

### ✅ Gerekli Durumlar

**1. Bayii Ağınız Varsa**
```
50+ bayii → B2B portal şart
Manuel sipariş → Otomasyon
Telefon/faks → Online sistem
```

**2. Özel Fiyatlandırma Varsa**
```
Her müşteri farklı fiyat
Sözleşme bazlı indirim
Yıllık anlaşmalar
```

**3. Vadeli Satış Yapıyorsanız**
```
30-60-90 gün vade
Kredi limiti takibi
Cari hesap yönetimi
```

**4. ERP Entegrasyonu Gerekliyse**
```
Stok senkronizasyonu
Otomatik fatura
Sipariş aktarımı
```

### ❌ Gerekli Değil Durumlar

**1. Sadece B2C Satış**
```
Son kullanıcıya satış
Tekrarlayan sipariş yok
Basit fiyatlandırma
```

**2. Küçük Ölçek**
```
Aylık ciro < ₺100K
Az sayıda müşteri
Manuel süreçler yeterli
```

**3. Tek Ürün/Hizmet**
```
Karmaşık katalog yok
Fiyat varyasyonu yok
Basit sipariş süreci
```

## B2B Maliyeti

### Hazır Platformlarda

```
B2B Modülü:
- İkas: ₺20K-₺50K/ay (üst paket)
- IdeaSoft: ₺30K-₺70K/ay
- Ticimax: ₺50K+ (tek seferlik + yıllık)
```

### Özel Yazılımda

```
B2B Paketi:
- Başlangıç: ₺150K-₺300K
- Yıllık bakım: ₺60K-₺120K
- Sunucu: ₺30K-₺60K/yıl
```

### ROI Hesabı

```
Senaryo: 100 bayii, aylık 500 sipariş

Manuel Süreç:
- Sipariş alma: 2 saat/gün
- Fiyat kontrol: 1 saat/gün
- Stok kontrol: 1 saat/gün
- Toplam: 4 saat/gün
- Maliyet: ₺20K/ay (personel)

B2B Portal:
- Otomatik sipariş
- Anlık stok
- Otomatik fiyat
- Maliyet: ₺5K/ay (sistem)

Aylık Kazanç: ₺15K
Yıllık Kazanç: ₺180K
+ Kaçan sipariş azalması: ₺100K+
```
