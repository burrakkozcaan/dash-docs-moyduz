# 3 Yıllık Sahip Olma Maliyeti (TCO) Hesaplayıcı

<div className="text-sm text-fd-muted-foreground mb-8">

E-ticaret platformu seçerken **sadece başlangıç maliyetine bakmak**, bir ev alırken sadece kapı fiyatına bakmak gibidir. **3 yıllık toplam maliyet (TCO)** gerçek resmi gösterir.

Bu rehberde, kendi senaryonuzla **gerçek maliyeti** hesaplayacaksınız.

## TCO Nedir ve Neden Önemli?

**Total Cost of Ownership (TCO)**, bir yatırımın satın alma fiyatından kullanım ömrü sonuna kadar olan **tüm maliyetlerini** içerir.

### E-Ticaret için TCO Kalemleri

```
TCO = Başlangıç + (Yıllık Tekrarlayan × Yıl) + Gizli Maliyetler + Fırsat Maliyetleri
```

## Kendi TCO'nuzu Hesaplayın

### Formül

```
HAZIR TCO = Başlangıç + Σ(Yıllık × (1+enflasyon)^yıl) + Gizli

ÖZEL TCO = Başlangıç + Σ(Yıllık × (1+0.05)^yıl)
```

### Değişkenler

| Değişken | Hazır | Özel |
|----------|-------|------|
| Enflasyon oranı | %50-100 | %5 |
| Komisyon | %0-2 | %0 |
| Tahsilat vadesi | 7-35 gün | 1-2 gün |
| Trafik limiti | Var | Yok |
| Modül ücretleri | Var | Yok |
