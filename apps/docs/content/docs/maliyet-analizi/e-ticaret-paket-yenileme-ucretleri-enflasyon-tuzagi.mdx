# E-Ticaret Paket Yenileme Ücretleri ve Enflasyon Tuzağı

İlk yıl **indirimli fiyat** ile başlayan e-ticaret yolculuğunuz, 2. yıl **%100-200 enflasyon zammı** ile kabusa dönüşebilir.

Bu rehber, hazır e-ticaret platformlarının **yenileme tuzağını** ve nasıl **korunacağınızı** anlatıyor.

## İlk Yıl İndirimi: Balık Yemi

### Nasıl Çalışır?

```
1. Yıl: ₺10.000/ay  (%60 indirim)
2. Yıl: ₺25.000/ay  (gerçek fiyat)
3. Yıl: ₺37.500/ay  (%50 enflasyon zammı)
```

**Gerçek:** İlk yıl indirimi, sizi **içeri çekmek** içindir.

### Sözleşme Maddesi Örneği

```
"Yenileme Fiyatlandırması:
Sözleşme yenileme döneminde, güncel liste fiyatları 
üzerinden %X enflasyon farkı eklenerek hesaplanır."
```

**Türkçesi:** "İstediğimiz fiyatı alırız, itiraz ederseniz sözleşmeniz biter."

## Enflasyon Tuzağı: Gerçek Örnekler

### Örnek 1: İkas

```
2024 Fiyatları:
- Plus:   $99/ay   → ₺3.300/ay
- Pro:    $249/ay  → ₺8.300/ay
- Enterprise: Özel

2025 Yenileme (TL bazlı):
- Plus:   ₺5.500/ay  (%67 artış)
- Pro:    ₺14.000/ay (%69 artış)
```

**Neden Dolar?** Dolar artışı + enflasyon = **çifte zam**

### Örnek 2: IdeaSoft

```
2024:
- Standart: ₺15.000/ay
- Pro:      ₺25.000/ay
- Enterprise: ₺45.000/ay

2025 Yenileme:
- Standart: ₺27.000/ay  (%80 artış)
- Pro:      ₺42.000/ay  (%68 artış)
- Enterprise: ₺75.000/ay (%67 artış)
```

### Örnek 3: Ticimax

```
İlk Yıl: ₺50.000 (tek seferlik) + ₺10.000/ay
2. Yıl:  ₺25.000 (yenileme) + ₺15.000/ay
3. Yıl:  ₺25.000 (yenileme) + ₺22.500/ay (%50 zam)
```

## Sözleşme Tuzakları

### 1. "Piyasa Koşulları" Maddesi

```
"Firmamız, piyasa koşullarına göre fiyat 
değiştirme hakkını saklı tutar."
```

**Anlamı:** "Haber vermeden zam yapabiliriz."

### 2. "Endeksleme" Maddesi

```
"Fiyatlar, TÜFE ortalaması + %20 oranında 
yıllık artırılır."
```

**Gerçek:** TÜFE %50 ise, zam %70 olur.

### 3. "Paket Değişikliği" Zorunluluğu

```
"Trafiğiniz paket limitini aşarsa, bir üst 
pakete geçmeniz zorunludur."
```

**Senaryo:**
- Başlangıç: 50K ziyaret (₺10K/ay)
- 12. ay: 80K ziyaret → Pro pakete zorla (₺30K/ay)
- **Zam: %200**

### 4. "Otomatik Yenileme"

```
"Sözleşme, aksi belirtilmedikçe otomatik 
1 yıl uzar."
```

**Tuzak:** İptal etmezseniz, otomatik yenilenir ve yüksek ücret ödersiniz.

## Gerçek Maliyet Hesaplama

### 3 Yıllık Proje (Orta Ölçek)

**İlk Yıl (İndirimli):**
```
Kurulum:      ₺25.000
Aylık (12):   ₺120.000 (₺10K/ay)
Modüller:     ₺30.000
───────────────────────
1. Yıl:       ₺175.000
```

**2. Yıl (Gerçek Fiyat + Enflasyon):**
```
Aylık (12):   ₺300.000 (₺25K/ay, %150 zam)
Modüller:     ₺50.000
Enflasyon:    ₺50.000
───────────────────────
2. Yıl:       ₺400.000
```

**3. Yıl (Ek Enflasyon):**
```
Aylık (12):   ₺450.000 (₺37.5K/ay, %50 zam)
Modüller:     ₺70.000
Enflasyon:    ₺75.000
───────────────────────
3. Yıl:       ₺595.000
```

**TOPLAM 3 YIL: ₺1.170.000**

### Özel Yazılım Karşılaştırma

**Özel Yazılım (Ticaret Paketi):**
```
Başlangıç:    ₺350.000 (sabit)
Yıl 1 Bakım:  ₺120.000
Yıl 2 Bakım:  ₺126.000 (%5)
Yıl 3 Bakım:  ₺132.300 (%5)
───────────────────────
TOPLAM:       ₺728.300
```

**Tasarruf: ₺441.700** (%38 daha ucuz)

## Korunma Yöntemleri

### 1. Sözleşmede Fiyat Sabitleme

```
"Fiyatlar 3 yıl boyunca sabittir, 
enflasyon zammı uygulanamaz."
```

**Not:** Çoğu platform kabul etmez.

### 2. Yenileme Öncesi İptal

```
- 60 gün önce yazılı bildirim
- Verileri export et
- Yeni altyapıya geç
```

**Risk:** Geçiş maliyeti ve operasyonel yük.

### 3. Özel Yazılıma Geçiş

```
- Tek seferlik ödeme
- Ömür boyu sahiplik
- Sadece sunucu + bakım (düşük, sabit)
```

**Avantaj:** Fiyat belirsizliği yok.

## Özel Yazılımda Durum

### Fiyat Şeffaflığı

```
Başlangıç:     ₺350.000 (sabit, yazılı)
Sunucu:        ₺5.000/ay (AWS faturası, şeffaf)
Bakım:         ₺10.000/ay (sabit, sözleşmeli)
───────────────────────────────────────
Yıllık Toplam: ₺180.000 (öngörülebilir)
```

### Enflasyon Koruması

```
- Başlangıç fiyatı sabit
- Sunucu maliyeti şeffaf (AWS dolar bazlı ama optimize edilebilir)
- Bakım anlaşması yıllık %5-10 artış (TÜFE bazlı)
```

## Ne Yapmalısınız?

### Sözleşme İmzalamadan Önce

1. [ ] 2. yıl yenileme fiyatını **yazılı** isteyin
2. [ ] Enflasyon zammı formülünü sorun
3. [ ] "Piyasa koşulları" maddesini sildirin
4. [ ] İptal şartlarını netleştirin
5. [ ] Veri export hakkını garanti alın
