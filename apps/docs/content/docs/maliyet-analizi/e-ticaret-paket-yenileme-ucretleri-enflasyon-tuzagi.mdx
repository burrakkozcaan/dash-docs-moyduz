İlk yıl **indirimli fiyat** ile başlayan e-ticaret yolculuğunuz, 2. yıl **%100-200 enflasyon zammı** ile kabusa dönüşebilir.

Bu karar rehberi, hazır e-ticaret platformlarının **yenileme tuzağını** ve bundan nasıl **korunacağınızı** interaktif hesaplamalarla anlatıyor.

## İlk Yıl İndirimi: Tüketici Psikolojisi Nasıl Yönetiliyor?

### Nasıl Çalışır?

```
1. Yıl: ₺10.000/ay  (%60 indirim)
2. Yıl: ₺25.000/ay  (gerçek fiyat)
3. Yıl: ₺37.500/ay  (%50 enflasyon zammı)
```

**Gerçek:** İlk yıl indirimi, sizi **içeri çekmek** içindir.

### Sözleşme Maddesi Örneği

```
"Yenileme Fiyatlandırması:
Sözleşme yenileme döneminde, güncel liste fiyatları 
üzerinden %X enflasyon farkı eklenerek hesaplanır."
```

**Türkçesi:** "İstediğimiz fiyatı alırız, itiraz ederseniz sitenizi kapatmak zorunda kalırsınız."

## Sözleşme Tuzakları ve Hukuki Riskler

<RiskAnalysis title="Bilinmeyen Sözleşme Maddeleri">
  **1. "Piyasa Koşulları" Maddesi:** "Firmamız, piyasa koşullarına göre fiyat değiştirme hakkını saklı tutar." Anlamı: Haber vermeden, yıl ortasında dahi hizmet bedeline ek (modül vb.) zam yapabiliriz.
  
  **2. "Endeksleme" Tuzağı:** "Fiyatlar, TÜFE ortalaması + %20 oranında yıllık artırılır." Gerçekte: TÜFE %50 ise, sistem size %70 zam yapma hakkını sözleşmeyle almıştır.
  
  **3. Otomatik Paket Yükseltme Zorunluluğu:** Başlangıçta ucuz paketi aldınız ancak trafiğiniz arttı. Sözleşmedeki madde gereği "Limit aşıldığında bir üst pakete geçiş zorunludur". Bu durumda zam oranınız %200'ü bulabilir.
  
  **4. Otomatik Kredi Kartı Yenilemesi:** "Sözleşme, iptal talebi 60 gün önceden ıslak imzalı iletilmedikçe otomatik kartınızdan çekilir." 
</RiskAnalysis>

## Gerçek TCO (Toplam Maliyet) Hesaplama

### 1. "Piyasa Koşulları" Maddesi

```
"Firmamız, piyasa koşullarına göre fiyat 
değiştirme hakkını saklı tutar."
```

**Anlamı:** "Haber vermeden zam yapabiliriz."

### 2. "Endeksleme" Maddesi

```
"Fiyatlar, TÜFE ortalaması + %20 oranında 
yıllık artırılır."
```

**Gerçek:** TÜFE %50 ise, zam %70 olur.

### 3. "Paket Değişikliği" Zorunluluğu

```
"Trafiğiniz paket limitini aşarsa, bir üst 
pakete geçmeniz zorunludur."
```

**Senaryo:**
- Başlangıç: 50K ziyaret (₺10K/ay)
- 12. ay: 80K ziyaret → Pro pakete zorla (₺30K/ay)
- **Zam: %200**

### 4. "Otomatik Yenileme"

```
"Sözleşme, aksi belirtilmedikçe otomatik 
1 yıl uzar."
```

**Tuzak:** İptal etmezseniz, otomatik yenilenir ve yüksek ücret ödersiniz.

## Korunma Yöntemleri ve Özel Yazılım

### 1. Sözleşmede Fiyat Sabitleme

```
"Fiyatlar 3 yıl boyunca sabittir, 
enflasyon zammı uygulanamaz."
```

**Not:** Çoğu platform kabul etmez.

### 2. Yenileme Öncesi İptal

```
- 60 gün önce yazılı bildirim
- Verileri export et
- Yeni altyapıya geç
```

**Risk:** Geçiş maliyeti ve operasyonel yük.

### 3. Özel Yazılıma Geçiş

```
- Tek seferlik ödeme
- Ömür boyu sahiplik
- Sadece sunucu + bakım (düşük, sabit)
```

**Avantaj:** Fiyat belirsizliği yok.
