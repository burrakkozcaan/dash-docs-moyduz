# Maliyet Analizi

E-ticaret platformu seçerken sadece başlangıç maliyetine değil, **3-5 yıllık toplam sahip olma maliyetine (TCO)** bakmak gerekir. Bu bölümde tüm gizli maliyetleri ortaya çıkarıyoruz.

## Maliyet Kalemleri

### Başlangıç Maliyetleri

| Kalem | SaaS | Özel Yazılım |
|-------|------|--------------|
| Kurulum | ₺5.000 - ₺50.000 | ₺100.000 - ₺500.000+ |
| Tema/Tasarım | ₺0 - ₺10.000 | ₺50.000 - ₺200.000 |
| Entegrasyonlar | ₺5.000 - ₺20.000 | ₺20.000 - ₺100.000 |
| Eğitim | ₺0 - ₺5.000 | ₺10.000 - ₺30.000 |
| **Toplam** | **₺10.000 - ₺85.000** | **₺180.000 - ₺830.000+** |

## Gizli Maliyetler

<Callout title="Dikkat">

Bu maliyetler genellikle sözleşmelerde belirtilmez!

</Callout>

### 1. Trafik Kotası Aşımı

Aylık ziyaretçi limitini aşarsanız:
- Ek paket ücreti: ₺2.000 - ₺20.000/ay
- Sayfa başına ücret: ₺0.01 - ₺0.05

### 2. Tahsilat Blokajı

Ödeme kuruluşlarında tutulan para:
- Ortalama blokaj: 2-7 gün
- Nakit akışı etkisi: Aylık cironun %5-15'i

### 3. Büyüdükçe Ceza

Bazı platformlar cironuz arttıkça:
- Komisyon oranını artırır
- Lisans ücretini revize eder
- "Enterprise" pakete zorlar

## 3 Yıllık TCO Karşılaştırması

| Yıl | SaaS Toplam | Özel Yazılım Toplam |
|-----|-------------|---------------------|
| 1. Yıl | ₺50.000 - ₺435.000 | ₺240.000 - ₺1.130.000 |
| 2. Yıl | ₺90.000 - ₺785.000 | ₺300.000 - ₺1.430.000 |
| 3. Yıl | ₺130.000 - ₺1.135.000 | ₺360.000 - ₺1.730.000 |

> **Kırılma Noktası:** Genellikle 2.5-3.5 yıl arasında özel yazılım daha ekonomik hale gelir.
