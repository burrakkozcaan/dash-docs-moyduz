# Sıfır Komisyon Yalanları ve Nakit Akışı

**"%0 komisyon!"** diyorlar. Ama tahsilatı **35 gün** içeride tutarak, gerçekte **%70-80 komisyon** alıyorlar.

Bu rehber, **görünmez komisyonun** gerçek maliyetini ve nakit akışınızı nasıl yok ettiğini açıklıyor.

## "%0 Komisyon" Nasıl Çalışır?

### Vaat

```
"Bizim platformumuzda %0 komisyon ile satış yapın!"
```

### Gerçek

```
Platform: "Komisyon %0"
Ama: "Tahsilat süresi ortalama 35 gün"
```

**Sorun:** Paranız 35 gün onların hesabında **bekliyor**.

## 35 Gün Vade: Gizli Komisyon Hesabı

### Senaryo

- Günlük satış: **₺50.000**
- Aylık ciro: **₺1.500.000**
- Tahsilat vadesi: **35 gün**

### Bloke Para

```
35 günlük satış: ₺50.000 × 35 = ₺1.750.000
```

**Bu para sizin ama erişemiyorsunuz.**

### Enflasyon Kaybı (Aylık %5)

```
Ay 1: ₺1.750.000 × %5  = ₺87.500  kayıp
Ay 2: ₺1.750.000 × %5  = ₺87.500  kayıp
Ay 3: ₺1.750.000 × %5  = ₺87.500  kayıp
────────────────────────────────────
3 Ay:                  ₺262.500  kayıp
```

### Fırsat Maliyeti

Bu ₺1.75M ile ne yapabilirdiniz?

```
- Yeni ürün stoğu:    %30 kâr marjı → ₺525.000 kâr
- Reklam yatırımı:    3x ROAS → ₺5.250.000 satış
- Nakit iskonto:      %5 → ₺87.500 indirim
```

## Gerçek Komisyon Oranı

### Hesaplama

```
Aylık ciro:         ₺1.500.000
Enflasyon kaybı:    ₺87.500
Fırsat maliyeti:    ₺150.000 (minimum)
────────────────────────────
Gizli maliyet:      ₺237.500/ay

Gerçek komisyon:    ₺237.500 / ₺1.500.000 = %15.8
```

**"%0 komisyon" = Gerçekte %15.8 komisyon**

### Ek Gizli Maliyetler

```
+ İşletme sermayesi ihtiyacı:    ₺50.000 (kredi faizi)
+ Stok yenileme gecikmesi:       ₺100.000 (kaçan satış)
+ Nakit akışı stresi:            (ölçülemez)
────────────────────────────────────
TOPLAM GİZLİ MALİYET:           ₺387.500/ay

GERÇEK KOMİSYON: %25.8
```

## Platform Karşılaştırması

| Platform | Vaat Edilen | Gerçek Vade | Gerçek Maliyet |
|----------|-------------|-------------|----------------|
| İkas | %0 | 35 gün | %18-25 |
| IdeaSoft | %0 | 14-30 gün | %8-15 |
| Ticimax | %0 | 7-21 gün | %5-12 |
| Shopify | 0.5-2% | 2-3 gün | %1-3 |
| **Özel Yazılım** | **%0** | **1-2 gün** | **%0-1** |

## Nakit Akışı Döngüsü

### 35 Gün Vade ile

```
Gün 1:   Satış yaptın (₺50K)
         ↓
Gün 35:  Parayı aldın (₺50K)
         ↓
Gün 35:  Stok yenile (₺30K)
         ↓
Gün 35:  Kalan: ₺20K (emeklilik vergi kira)
```

**Sorun:** 35 gün boyunca **yeni stok alamıyorsunuz**.

### 1 Gün Vade ile

```
Gün 1:   Satış yaptın (₺50K)
         ↓
Gün 2:   Parayı aldın (₺50K)
         ↓
Gün 2:   Stok yenile (₺30K)
         ↓
Gün 2:   Yeni stokla satışa devam
```

**Fark:** **35x daha hızlı** stok devir hızı.

## Gerçek Örnek: KOBİ Senaryosu

### Mehmet Bey - Elektronik Satıcısı

**Durum:**
- Aylık ciro: ₺800.000
- Kâr marjı: %20
- Platform: İkas (35 gün vade)

**Sorun:**
```
Ocak satışları: ₺800.000
Para eli geçti: 5 Mart (35 gün sonra)
Şubat stok alamadı: Kaçan satış ₺400.000
Kredi çekti: ₺300.000 (%5 faiz/ay)
```

**Sonuç:**
```
Ocak kârı:          ₺160.000
Kredi faizi:        -₺45.000
Kaçan satış kârı:   -₺80.000
────────────────────────────
NET:                ₺35.000
```

**Gerçek kâr marjı: %4.4** (>%20 yerine)

### Aynı Senaryo - 1 Gün Vade

```
Ocak kârı:          ₺160.000
Kredi faizi:        ₺0
Kaçan satış:        ₺0
────────────────────────────
NET:                ₺160.000
```

**Fark: ₺125.000/ay = ₺1.5M/yıl**

## Çözüm: Ertesi Gün Tahsilat

### Özel Yazılımda

```
Ödeme Altyapısı:
- iyzico, PayTR, Param
- Doğrudan sizin hesabınıza
- T+1 (ertesi gün) tahsilat
- Hafta sonu dahil
```

### Nakit Akışı

```
Günlük satış:  ₺50.000
Haftalık:      ₺350.000 (her Pazartesi hesabınızda)
Aylık:         ₺1.500.000

Stok devri:    7 gün (haftada 1)
Yeni yatırım:  Sürekli
```

## Komisyonlu Model Karşılaştırması

### Trendyol Modeli

```
Komisyon:       %15 (elektronik)
Tahsilat:       30-45 gün
Kargo:          Alıcı öder veya ₺30-50
Reklam:         Zorunlu (ciro %5-10)
────────────────────────────
TOPLAM MALİYET: %25-35
```

### Hazır SaaS

```
Komisyon:       %0 (vaat)
Tahsilat:       35 gün
Enflasyon:      %15 (35 gün)
Fırsat:         %10 (stok devri)
────────────────────────────
TOPLAM MALİYET: %25
```

### Özel Yazılım

```
Komisyon:       %0
Tahsilat:       1 gün
Enflasyon:      %0.5 (1 gün)
Sunucu:         ₺5.000/ay (sabit)
────────────────────────────
TOPLAM MALİYET: %0-1
```

## 3 Yıllık Nakit Akışı Kaybı

### Senaryo: Aylık ₺1M ciro, %20 marj

**35 Gün Vade:**
```
Yıl 1: ₺1.75M × %5 × 12 = ₺1.05M kayıp
Yıl 2: ₺2.5M × %5 × 12  = ₺1.5M kayıp
Yıl 3: ₺3.5M × %5 × 12  = ₺2.1M kayıp
────────────────────────────
TOPLAM: ₺4.65M kayıp
```

**1 Gün Vade:**
```
Yıl 1: ₺50K × %5 × 12 = ₺30K kayıp
Yıl 2: ₺75K × %5 × 12 = ₺45K kayıp
Yıl 3: ₺100K × %5 × 12 = ₺60K kayıp
────────────────────────────
TOPLAM: ₺135K kayıp
```

**Fark: ₺4.515.000**
